This Web Application implements a Machine Learning solution to classify News Articles into 5 categories - Business, Entertainment, Politics, Sport and Technology. This application utilizes state-of-the-art transformer models to process the news article and classify it into its correct category. The demand for information by users has been growing continuously, so it is crucial that the news is classified to allow users to access the information of interest quickly and effectively. This way, the machine learning model for automated news classification could be used to identify topics of untracked news and/or make individual suggestions based on the user's prior interests.

The Web Application can be utilized to Categorize a single News Article or multiple articles submitted in a single .csv file.

To classify News Articles upload a .csv (Comma Separated Value) file in the following format

    1. First Column Heading: "ArticleId"
    2. Second Column Heading: "Text"
  • The data to be predicted on should start from 2nd row
